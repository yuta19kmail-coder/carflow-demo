💾 データ管理

バックアップ・一括編集・初期化

📦 完全バックアップ（JSON）

在庫車・販売実績・付箋・設定・テンプレを丸ごとJSONで書き出し／取り込みできます。
万一に備えたバックアップ用途。LINE設定とスタッフは含みません。
※ 写真本体（Storage）はバックアップ対象外です（URL/参照のみ含まれます）

📊 車両一括編集（Excel）

在庫車を Excel で書き出し、編集してから取り込み直せます。
編集対象：管理番号 / メーカー / モデル / 年式 / 色 / サイズ / 走行距離 / 価格 / 仕入日 / 納車予定日 / 状態 / お客様名 / メモ / 作業メモ
状態の選択肢：その他 / 仕入れ / 再生中 / 展示中 / 納車準備 / 納車完了
触らない：チェック状況・装備品・写真・操作ログ（既存のまま保持）
ID列は編集禁止。空欄行は新規登録扱いになります。

🗑 全消去

車両・販売実績・付箋ボードを全て削除します（設定・テンプレ・スタッフ・LINE連携は残ります）。
⚠️ この操作は元に戻せません。事前にバックアップを取ってから実行してください。

📦 写真の Storage 移行

v1.5.10 より前にアップロードされた data:URL 形式の写真／アイコンを Firebase Storage に一括移行します。Firestore の容量を節約できます。
