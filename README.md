# CarFlow デモ版

中古車整備工場・販売店向け業務管理ツール **CarFlow** の体験版です。

本番URL：https://carflow.kobayashi-motors.com

## 🎯 デモ版の特徴

- **ログイン不要**：ボタン1クリックで即起動
- **サンプルデータ入り**：在庫車20台、過去24ヶ月の販売実績、付箋メモ等が最初から入ってる
- **DB不使用**：Firestore に繋がない、ブラウザのメモリだけで動作
- **保存されない**：リロードすると最初の状態に戻る
- **LINE通知はプレビュー**：実際のLINEには送られず、画面右下にスマホ風の吹き出しで「こんなメッセージが届きます」を表示
- **全機能触れる**：車両追加・編集・タスク完了・付箋追加・月次集計など、すべて自由に操作OK

## 🚀 起動方法

`index.html` をブラウザで開くだけ。

または GitHub Pages 等の静的ホスティングに上げれば誰でもアクセス可能。

## 📂 デモ専用ファイル（追加分）

| ファイル | 役割 |
|---|---|
| `js/demo-firestore-mock.js` | Firestore SDK の in-memory モック |
| `js/demo-storage-mock.js`   | Storage SDK の in-memory モック |
| `js/demo-init.js`           | `window.fb` をモックで初期化 |
| `js/demo-auth-mock.js`      | ログイン画面を「デモを始める」1ボタンに改造 |
| `js/demo-line-toast.js`     | LINE通知を画面右下のスマホ風UIで表示 |
| `js/demo-sample-data.js`    | 起動時にサンプルデータを投入 |
| `css/demo.css`              | 上部バナー＋トースト用スタイル |

## ⚠️ 注意

- 本番（実運用版）と同じコードベース上に**置換ファイルだけ追加**したものです
- 本番には触れない設計（別フォルダで完結）
- このデモ版には個人情報や実データは含まれていません（全部ダミー）
